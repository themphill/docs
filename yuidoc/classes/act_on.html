Checks to see if the integration data is present, then finds and dispatches that data The integration's data is stored both in cookies and also in the client code

The nature of the data is unique to each integration, but should always be wrapped in an object. The backend integration code is responsible for parsing that data. The wrapping if() block should not be modified.

    Must be passed to dispatchData()
