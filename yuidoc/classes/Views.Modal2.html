Views.Modal2 Class

Defined in: ../../../application/modules/Carbon/resources/js/views/view.modal2.js:1
Module: Carbon

A better modal than the first

This class defines init, render, and unrender in a way that is dramatically better than the implementation in Ibp.Carbon.Views.Modal. That class is deprecated. Don't use it.

Views that inherit from this one only need to implement the onInit(), onRender(), and onUnrender() hooks to add additional functionality. The modal's parent view should then call show() to display the modal. The modal's template must include the root modal element.
