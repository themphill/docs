Views.SnippetModal Class

Defined in: ../../../application/modules/sourcetrak/resources/js/views/view.snippetModal.js:1
Module: SourceTrak

St Snippet Modal Modal that displays the ST Snippet with Copy to Clipboard button

This class defines init, render, and unrender in a way that is dramatically better than the current modal implementation in Ibp.Carbon.Views.Modal. That base class should be refactored like this one. Views that would inherit from the refactored base class would only needs to implement the onInit(), onRender(), and onUnrender() hooks to add additional functionality. The modal's parent view would then call show() instead of render(). The modal's template must include the root modal element, instead of creating it in init(), as in the template here.
