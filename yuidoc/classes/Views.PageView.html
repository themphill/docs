showSelectInput

()

Provided by the Sourcetrak module.

Defined in ../../../application/modules/sourcetrak/resources/js/views/view.mainNumberSelect.js:90

Shows the edit select input This is overriding the default showSelectInput function in Ibp.Carbon.Views.EditableSelect specifically so that we can only build the select element only when it is needed. Such as the first time the user attempts to interact with the editable control. This allows us to avoid building it on page load, in order to avoid page hang when crunching thousands of main numbers.
