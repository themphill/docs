Mixins.Transform Class

Defined in: ../../../application/modules/Carbon/resources/js/mixins/model.transform.js:1
Module: Carbon

Mixin for Models that allows them to cast values being set on them to the appropriate type Its primary purpose is to handle converting string values from HTML form inputs

Transforms are done by matching the property being set on the model to to a transform method specified in the model's 'transforms' object

Transform relies on a modified version of DeepModel.set() and only works with DeepModels
